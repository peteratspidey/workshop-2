# python basics
## string data type
### declare a string
```python
my_string = "hello vivek"
```
### check the type of the string
### len function
```python
len("my_string")
```
### replacing the string
```python
string.replace("vivek", "peter")
```
### indexing and slicing
```python
print(my_string[0])
print(my_string[5])
print(my_string[0:3])
print(my_string[0:3:2])
print(my_string[-1:-4])

```
